# Security Policy

This repository is an automatically generated **public mirror** of the changelog for the
[davesandersofficial.com](https://davesandersofficial.com) web application. It contains no
application source code or secrets — only release metadata (`CHANGELOG.md`, `manifest.json`)
and tagged GitHub Releases.

## Reporting a vulnerability

Please **do not** open a public issue. Report any security concern — whether about this
repository or the application itself — privately to:

**[info@davesandersofficial.com](mailto:info@davesandersofficial.com)**

We aim to acknowledge reports within a few business days.
